# BCC-website

## 📖 Description
BCC Educational Website is a simple multi-page school website created using HTML and CSS.  
It is designed to provide information about the school, courses, teachers, departments, and contact details.

---

# ✨ Pages Included

- Home Page
- About Page
- Courses Page
- Teachers Page
- Department Page
- Contact Page

---

- HTML
- CSS
