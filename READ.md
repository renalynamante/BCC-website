# BCC Website

## Overview
BCC Website is a simple educational website built using HTML and CSS only.  
The project contains multiple pages such as Home, About, Courses, Teachers, Department, and Contact.

---

# Features

- Responsive website layout
- Navigation bar
- Hero section
- Courses section
- Teachers section
- Department section
- Contact form
- External CSS support

---

# Technologies Used

- HTML5
- CSS3
